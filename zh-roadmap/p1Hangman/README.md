# Hangman with console UI using ASCII characters

## Функционал
- При старте предлагает начать новую игру или выйти из приложения.
- При начале новой игры загадывается случайное слово, игрок начинает процесс отгадывания.
- После введения каждой буквы в консоль выводится количество ошибок и текущее состояние виселицы.
- В конце игры выводится результат (выигрыш/проигрыш) и процесс возвращается к стартовому приглашению.

## Реализация
ASCII образы состояний находятся в папке resources и пронумерованы от 0 (нет ошибок) до 6 (предельное кол-во ошибок).
